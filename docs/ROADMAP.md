# UNI-NEST Development Roadmap

The project is built in 15 phases. Each phase must be fully completed, tested,
and documented before moving to the next.

1. **Project Initialization** — folder structure, Next.js, Express, Prisma, tooling
2. **Database Design** — ER diagram, Prisma models, migrations, seed data
3. **Authentication System** — register, login, JWT, Google login, RBAC, email verification
4. **Client Portal** — landing page, listings, wishlist, reservations, profile, reviews
5. **Property Search** — filtering, sorting, pagination, maps, nearby universities
6. **Reservation System** — reserve, viewings, calendar, status, email notifications
7. **Landlord Portal** — dashboard, property CRUD, media, analytics, staff
8. **University Portal** — housing info, recommended accommodation, statistics
9. **Company Portal** — real estate & property management dashboards
10. **Admin Portal** — user management, approvals, reports, monitoring
11. **Communication System** — real-time chat, notifications, voice notes
12. **AI Features** — smart search, recommendations, moderation, chatbot
13. **Testing** — auth, reservations, search, CRUD, validation, security, roles
14. **Documentation** — README, API docs, guides, manuals
15. **Deployment** — Vercel (frontend), Render (backend), Railway (MySQL), Supabase (storage)
